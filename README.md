Requirements
============
Java 1.7 or later.  

Installation
============

前往 [契约锁开放平台](http://open.qiyuesuo.com/download)下载Java SDK及依赖包，并添加到项目中。

### Maven users

将下载的Java SDK上传至Maven私有仓库，或本地仓库，并在项目POM文件中添加:

```xml
<dependency>
	<groupId>com.qiyuesuo.sdk</groupId>
	<artifactId>sdk-java</artifactId>
	<version>3.0.0</version>
</dependency>
```
